<h1 align="center">Welcome to tiktaktoe 👋</h1>

This is a very simple project that shows up how to stub external service calls in unit testing.

## Install dependencies
```bash
npm install
```

## Build
```bash
npm run build
```

## Start
```bash
npm run start
```

## Run Unit testing
```bash
npm run test
```
